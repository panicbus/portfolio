    Scenic 49

     Built with: Ruby on Rails, HTML5, CSS3, JavaScript, Isotope.js, MapBox, AWS, Paperclip Gem



    Map View with Station Markers



    All Checkins Page



    Mobile view

    On The Road: The Scenic 49 is the interactive online map component of an art project developed by Peter Foucault and Chris Treggiari.

    The app uses the MapBox API for the mapping functionality, Amazon Web Services for photo storage, and the Paperclip Ruby Gem for the photo uploading protocol.

    This project can be cloned and reproduced for any city in the world (click 'next' below). Feel free to head over to Github and fork it.

    On Camera

     Built with: Ruby on Rails, HTML5, CSS3, JavaScript, Isotope.js, MapBox, AWS, Paperclip Gem



    Map View with Location Markers



    All Checkins Page



    Mobile splash view

    On Camera: The Watchmen is a cloned iteration of the Scenic 49 project, the interactive online map for another Peter Foucault and Chris Treggiari project.

    This version uses the exact techonology as the Scenic 49 version, only having changed the map and using new GeoJson data for each location.

    This project is also open-source and available to be forked. It should less than 1 hour to convert to any city of your choosing.

    WiFly

     Built with: Ruby on Rails , HTML5 , CSS3 , Bootstrap



    WiFly Main

    WiFly lets you to search for airports anywhere around the world to determine whether they have an available WiFi signal for public Internet use, if the signal is free or the fee to use it. Most data is crowdsourced. Currently in beta.

    The Bootstrap CSS framework is in use here, on top of a Ruby on Rails baselayer. A map feature is being developed for version 2.0.
