    Treasure Hunter

    Ruby on Rails | HTML5 | CSS3 | JS | CoffeeScript | GoogleMaps & Twilio Api | Devise




    Treasure Hunter Map Page



    Hunt List



    Mobile









    Treasure Hunter is a urban scavenger hunt game played using your mobile device and the streets of your city! Players can be Huntmasters (to create hunts for others) or Hunters (upon invitation) to get clues and find secret locations around the town. Get your friends together and Treasure Hunt! Currently in alpha.

    WiFly

    Ruby on Rails | HTML5 | CSS3 | Bootstrap




    WiFly Main









    WiFly lets you to search for an airport anywhere around the world to see if it has an available WiFi signal for public use, if it's free, and if it's not, what's the fee to use it. Most data is crowdsourced. Currently in beta.
